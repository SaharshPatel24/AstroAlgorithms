Clone and Use the Repository

Follow these steps to clone and use the repository:

  1. Clone the Repository: Open your terminal and run the following command to clone the repository:

    Example :
    git clone https://github.com/SaharshPatel24/AstroAlgorithms.git

  2. Install Dependencies: Navigate into the cloned repository folder and install dependencies:

    Example :
    cd your-repo
npm install

  3. Setup Environment Variables: Set up your environment variables by creating a .env file and adding necessary configurations. Refer to the provided .env.example for required variables.

  4. Run the Application: Start the backend server by running:

    Example :
    npm run start

  5. Explore API Endpoints: Visit Swagger API Documentation to explore and interact with the available API endpoints.
